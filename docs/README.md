- 成長するために、維持よりも変化し続けることを望んでいます。変化や困難はモチベーションになります。
- 少人数で一人一人の責任が重いプロジェクトだとモチベーションが上がります。

- 課題を解決していること、誰かのためになっていることに喜びを感じます。
