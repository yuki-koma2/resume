# yuki komatsu Resume

転職サイトだと、自分の思うように書けないので、自由に書き定期的に更新するように
githubで書いて管理します。

## Development


### Setup

```shell
$ git clone
$ cd resume
$ npm install
```

### lint

文章チェック

```shell
npm run lint
```

自動修正

```shell
npm run lint:fix
```
